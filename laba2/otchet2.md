## Задание 1

### Условие:

Тимофей составляет 5-буквенные коды из букв Т, И, М, О, Ф, Е, Й. Буква Й может использоваться в коде не более одного раза, при этом она не может стоять на первом месте, на последнем месте и рядом с буквой И. Все остальные буквы могут встречаться произвольное количество раз или не встречаться совсем. Сколько различных кодов может составить Тимофей?

### Описание проделанной работы

Генерация всех возможных кодов. С помощью itertools.product создаются все возможные 5‑буквенные комбинации из букв «ТИМОФЕЙ» (с повторениями).
Проверка условий для каждого кода:
Не более одной буквы «Й»: проверяется через p.count('Й') <= 1.
«Й» не на первом месте: условие p[0] != 'Й'.
«Й» не на последнем месте: p[-1] != 'Й'.
«Й» не рядом с «И»: проверяются подстроки 'ИЙ' и 'ЙИ' через p.count('ИЙ') == 0 и p.count('ЙИ') == 0.
Подсчёт допустимых кодов. Если код удовлетворяет всем условиям, счётчик k увеличивается на 1.
Вывод результата.

## Задание 3

### Условие: 
Найдите среди целых чисел, принадлежащих числовому отрезку 
[174457;174505] числа, имеющие ровно два различных натуральных делителя, не считая единицы и самого числа. 
Для каждого найденного числа запишите эти два делителя в два соседних столбца на экране с новой строки в порядке возрастания произведения этих двух делителей. 
Делители в строке также должны следовать в порядке возрастания. 
Например, в диапазоне [5;9] ровно два различных натуральных делителя имеют числа 6 и 8, поэтому для этого диапазона вывод на экран должен содержать следующие значения: 23 24

### Описание проделанной работы: 

Код перебирает числа из диапазона [174457; 174505], находит для каждого числа делители (кроме 1 и самого числа), отбирает числа с ровно двумя делителями, сортирует их по произведению делителей и выводит пары делителей в порядке возрастания.
