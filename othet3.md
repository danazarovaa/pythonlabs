## Задание 2

### Условие: 

Написать функцию f(n), которая рекурсивно подсчитывает общее количество несписковых элементов (чисел, строк и других атомарных значений) в структуре списка, который может содержать вложенные списки произвольной глубины.

### Описание проделанной работы: 

Реализована рекурсивная функция для подсчёта элементов в вложенных списках. Функция обходит каждый элемент списка: если элемент — список, вызывает себя; если нет — увеличивает счётчик. Тестирование на примере ["x", "y", "z" [45]] подтвердило корректность: результат — 4. Работа выполнена без использования глобальных переменных и итеративного стека.

## Задание 3

### Условие: 

Разработать функцию f(i), вычисляющую i-й элемент числовой последовательности, заданной рекуррентной формулой:

x_i = ((i−1)⋅(x_i−1))/3+((i−2)⋅(x_i-2))/4

с начальными условиями:

x_1 =1, x_2=−0.125

### Описание проделанной работы: 

Реализована итеративная функция для вычисления x_i по заданной рекуррентной формуле. Для i = 1 и i = 2 возвращаются начальные значения. Для i ≥ 3 используется цикл, в котором последовательно вычисляются значения с сохранением двух предыдущих результатов. Тестирование при i = 5 дало корректный результат.
